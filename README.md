## Usage
```
Usage: macos-log-dfir timeline [OPTIONS] --output <OUTPUT>

Options:
  -h, --help  Print help

Input:
  -a, --archive_dir <ARCHIVE>  Path to logarchive formatted directory
  -l, --live_analysis          Run on live system

Output:
  -o, --output <OUTPUT>
```
